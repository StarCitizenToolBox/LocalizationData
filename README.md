## 手动安装
1. 前往 [TAGS](https://jihulab.com/StarCitizenCN_Community/localizationdata/-/tags) 页面，选择最新的版本，点击下载源代码（zip）。
2. 解压zpi，将 chinese_(simplified)文件夹 放入游戏 Live/data/Localization/ （没有则创建对应文件夹）
